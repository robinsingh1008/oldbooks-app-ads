# Old Books App - App Ads Setup

This folder contains the files needed to host your `app-ads.txt` file for AdMob verification.

## Files in this folder:
- `app-ads.txt` - The AdMob verification file
- `index.html` - Simple homepage (optional)
- `README.md` - This file

## Quick Deploy Instructions

### For GitHub Pages:
1. Create a GitHub repository
2. Upload this entire folder
3. Enable GitHub Pages in Settings
4. Your file will be at: `https://YOUR-USERNAME.github.io/oldbooks-app-ads/app-ads.txt`

### For Netlify:
1. Go to https://app.netlify.com
2. Drag this folder to Netlify
3. Done! Your file will be live in 30 seconds

### For Firebase Hosting:
1. Run: `firebase init hosting`
2. Run: `firebase deploy`
3. Your file will be at: `https://YOUR-PROJECT.web.app/app-ads.txt`
